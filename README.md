- **Python:** 3.13.7

### Установка проекта

Установите зависимости:

```bash
pip install -r requirements.txt
````
Комментарии по багам:

В ходе тестирования были обнаружены следующие баги API, которые помечены в коде как @pytest.mark.xfail:

Создание курьера: При попытке создания дубликата или запроса с существующим логином сервер возвращает некорректное сообщение.

Логин курьера: При передаче пустого поля пароля или неверных данных система может возвращать 504 вместо 400 Bad Request.

Создание заказа: Проверено параметризацией для различных цветов (BLACK, GREY, оба, ни одного).

````bash

pytest -v -s
````
